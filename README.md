## Guia de Operação
Conecte o Arduino, abra o Monitor Serial (baud 9600) e envie:
- **U**: Move para cima (Up)
- **D**: Move para baixo (Down)
- **O**: Abre a garra
- **C**: Fecha a garra

## Especificações Técnicas
- **Fonte**: 6V via fonte de bancada simulada.
- **Pinagem**: Servo Arm (Pino 9), ServoGarra (Pino 10), LED Status (Pino 13).
- **Software 3D**: Fusion 360.
